# Larderlab — Topical Map

5 hubs × (1 pillar + 3 comparison/calculator + 18 cluster + ~10 listicle) ≈ 160 posts. Wave 1 = 5 pillars + 6 priority comparisons/calculators + 29 cluster posts (~40 total). Waves 2-4 fill in the rest.

**Strategic note:** Larderlab has less Pinterest upside than plasticfreelab, so listicles are fewer and denser. Calculators + frameworks replace a portion of what would be listicle slots in other sites. These are treated as pillar-adjacent content.
